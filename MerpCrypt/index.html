Description:

  • 🔐 MerpCrypt is a creative encryption-decryption tool built using Caesar Cipher logic — with a twist of fun using emojis!
  • 🔢 PIN-Protected: You set a 4-digit PIN, and that becomes your encryption key. No PIN = no access to the original message.
  • 😀 Emoji Magic: Instead of encrypted letters, your message gets turned into an expressive string of emojis. Every letter maps to an emoji — making the output fun and secure at the same time.
  • 🎮 Simple & Interactive: Clean user interface with easy-to-use buttons for encrypting, decrypting, and copying results instantly.
  • 🧠 Educational Touch: Behind the scenes, it uses Caesar Cipher logic — great for anyone curious about how encryption works at a basic level.
  • 📱 Lightweight & Responsive: Designed to work smoothly on desktop — no downloads, no setup, just direct fun in the browser.
  • 🎨 Aesthetic + Functional: Modern UI with a touch of creative madness (yeah, emojis everywhere) 😄
  • 💡 My Vision: I created MerpCrypt to blend coding, security, and fun — so anyone, even without technical background, can play with encryption and enjoy the process.
  • 👨‍💻 Built with Passion: From scratch, designed, UI, Idea, and everythings holds is developed by me — Prem, a student and tech enthusiast from CEG, Anna University.
